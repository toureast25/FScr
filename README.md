# FScr — Fast Screenshot

В ходе изучения работы с СКД требовалость предавать проставленные параметры из конфигуратора в поисковый запрос.
Искать в интернете я предпочитаю через ИИ, а значит достаточно кормить его скриншотами. Стандартные Ножницы - долго возиться, 
PrtScr - тянет лишнее с экрана.
Результат: навайбкодил такой инструмент.

## Возможности

- Захват области экрана с помощью выделения курсором
- Горячая клавиша **Ctrl+Alt+S**
- Автоматическое копирование в буфер обмена (PNG)
- Работает в системном трее

## Установка

### Скачать готовый релиз

Скачайте `FScr.exe` из [Releases](https://github.com/toureast25/FScr/releases) и запустите.

### Сборка из исходников

```bash
git clone https://github.com/toureast25/FScr.git
cd FScr
pip install -r requirements.txt
python FScr.py
```

## Сборка exe

```bash
pip install -r requirements.txt
pip install pyinstaller
python -m PyInstaller --onefile --noconsole --name "FScr" FScr.py
```

Или используйте `build.bat`.

## Использование

1. Запустите `FScr.exe`
2. Нажмите **Ctrl+Alt+S** или кликните на иконке в трее
3. Выделите область экрана
4. Скриншот автоматически скопируется в буфер обмена
5. Отмена по ESC или ПКМ


## Системные требования

- Windows 10/11
- Python 3.8+ (для сборки)

## Зависимости

- mss — захват экрана
- Pillow — обработка изображений
- pywin32 — буфер обмена
- pystray — системный трей
- keyboard — горячие клавиши
